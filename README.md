## Simple Portfolio Website

A clean, responsive portfolio website, entirely created by an AI Agent for demonstration purposes.

Live demo:
https://lawson-han.github.io/simple-portfolio-website/

Features:

- Responsive layout for desktop, tablet, and mobile
- Light/Dark theme toggle with localStorage persistence
- Smooth scrolling navigation
- Accessible, keyboard-friendly UI (semantic HTML, ARIA, clear focus states)

Tech stack:

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans, JetBrains Mono)

Project structure:

- index.html
- styles.css
- script.js
- images/
  - personal/
  - logos/
- README.md

Getting started:

Clone:
```
git clone https://github.com/lawson-han/simple-portfolio-website.git
cd simple-portfolio-website
```

Run:

- Open index.html directly, or
- Start a local server:
  - python -m http.server 8000
  - or npx http-server
